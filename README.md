To run LLaVA-Med model, clone repo below into the root directory.
```
https://github.com/microsoft/LLaVA-Med
```

To run LLaVA-Med fine tuned model, kindly download model file from kaggle.
```
https://www.kaggle.com/datasets/chaijunyi/qlora-llava-med-merged-model
https://www.kaggle.com/datasets/slayerjunyi/dora-llava-med-merged-model
```
Put all the file under finetune/qlora-llava and finetune/dora-llava directory.
